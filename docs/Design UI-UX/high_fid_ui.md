# High-Fidelity UI (Mockup)

---

## Karakter Visual Umum 
### Skema Warna
- Background utama: Abu-abu muda (#A3A3A3)
- Primary color: Hijau tua kebiruan (#005461) → tombol utama
- Secondary color: Putih (#FFFFFF) → form & card
- Aksen: Hijau checklist & ikon sukses
- Teks utama: Hitam & abu gelap
Warna memberi kesan tenang, modern, dan profesional, cocok untuk coffee shop.

### Tipografi
- Judul halaman: Serif dekoratif (gaya logo ONE Love)
- Body text & form: Sans-serif sederhana (mudah dibaca)
- Ukuran teks:

    Judul halaman: ±24–28 px

    Subjudul: ±18 px
  
    Isi & label: ±14–16 px

### Layout & Grid
- Layout single column dan centered
- Header konsisten di setiap halaman:

    Logo ONE Love di kanan atas

    Judul halaman di kiri
- Tombol navigasi selalu di bagian bawah
- Spacing luas → nyaman di mobile & desktop

## Deskripsi High-Fidelity UI per Halaman
### Landing Page
#### Tampilan:
- Ilustrasi cangkir kopi di sisi kanan
- Headline:
“Rasakan Nikmatnya Kopi Hanya di One Love Coffee”
- Tombol SIGN IN di kanan atas
#### Tujuan UI:
- Menarik perhatian
- Mengarahkan user ke login

Konsisten dengan requirement akses awal sistem

### Login & Register
#### Komponen:
- Card putih rounded
- Field:

    Username / Email

Password
- Tombol:

    LOG IN

    Register
- Opsi remember me & forgot password
#### Prinsip usability:
- Form sederhana
- Fokus pada satu aksi utama

Konsisten dengan FR-01 (Autentikasi User)

### Daftar Menu (Makanan & Minuman)
#### Tampilan:
- Menu ditampilkan dalam card
- Setiap card berisi:

    Gambar produk

    Nama menu

    Harga
#### Tombol:

(+) tambah jumlah

ikon keranjang
- Navigasi Back & Next
#### Kategori:
- Makanan
- Minuman

Konsisten dengan User Journey – Browse & Select Menu

### Halaman Pesanan
#### Isi halaman:
- Tabel:

    Nama Produk

    Jumlah

    Harga
- Ringkasan Total Harga
- Tombol:

    Back

Mendukung checkout clarity & error prevention

### Metode Pembayaran
#### Pilihan:
- Transfer Bank (BCA, BNI, Mandiri, BRI)
- E-Wallet (GoPay, OVO, Dana, ShopeePay)
- Pembayaran Langsung (Bayar di Tempat)

Mengakomodasi berbagai preferensi user
